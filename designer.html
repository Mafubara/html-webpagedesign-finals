About the Designer

The creative mind behind Ximi Guides

Naven Cuenca

Web Designer & Developer

Hello! I'm the designer and developer behind Ximi Guides. Creating this website was both challenging and rewarding as I navigated through HTML, CSS, and JavaScript to bring this project to life.

As someone who's still learning web development, building Ximi Guides taught me valuable lessons about:

  • Responsive design principles
  • CSS Grid and Flexbox layouts
  • JavaScript interactivity
  • Maintaining consistent styling

The most surprising lesson? How much work goes into making things look simple!
